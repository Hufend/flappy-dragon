Flappy Dragon 是一个 **Flappy Bird** 的简单 Rust 实现

这是一个非常简单的小游戏, 你只需要按下空格让小龙 `@`  往上飞, 使小龙不撞到障碍物即可得分

与好友比拼一下谁的得分多吧!
